I wanted to document the decisions behind building this site, both as a reference for myself and for anyone curious about putting together a modern personal website.

## The Stack

Here's what's under the hood:

- **Next.js** — React framework with server-side rendering, static generation, and the App Router
- **Tailwind CSS v4** — Utility-first CSS with the new alpha engine
- **MDX** — Markdown with embedded React components for blog posts
- **Geist** — Vercel's beautiful font family for a clean, modern look
- **sugar-high** — Lightweight syntax highlighting for code blocks

## Why This Approach?

I wanted something that felt fast, looked polished, and was easy to write for. MDX is perfect for blogging — I get the simplicity of Markdown with the flexibility of React when I need it.

The file-system based routing in Next.js means adding a new blog post is as simple as dropping a `.mdx` file into the posts directory. No CMS, no database — just files.

## Code Highlighting

Here's how the syntax highlighting looks with `sugar-high`:

function greet(name) {

Clean and readable, without pulling in a heavy library.
